Bakchod MusicApp – Quick Personal Steps
1️⃣ Open Project Folder

Go to your BakchodMusicApp folder.

2️⃣ Make Sure Python is Installed

Python 3.10+

python --version

3️⃣ Install Dependencies
pip install -r requirements.txt

4️⃣ Check FFmpeg

ffmpeg.exe + ffprobe.exe → inside ffmpeg/ folder

Path in app.py:

FFMPEG_PATH = r"ffmpeg\ffmpeg.exe"

5️⃣ Run the App
python app.py

6️⃣ Open in Browser

Go to URL from terminal:

http://127.0.0.1:5000

7️⃣ Download a Song

Copy YouTube link

Paste in input box

Click Download MP3

Wait for progress bar → 100%

8️⃣ Play a Song

Click song in list → plays in built-in player

9️⃣ Manage Songs

Delete → remove from songs/ folder

Add more → repeat step 7
